# Growio

A plant identifier,  tracker, and plantcyclopedia.

## About Our App
This project is an Android/iOS app that aims to help you manage your plants
 and learn more about them in the process!

## Features
* Take a picture of your plant to find out more about it!
* Manage all of your plants in MyGarden!
* Find out more about plants by consulting the Plantcyclopedia!
